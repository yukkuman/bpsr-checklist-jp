# ブルプロ：デイリー・ウィークリーチェックリスト

Maxroll の「Dailies and Weeklies Guide」をベースに、日課・週課・週替わりショップを日本語チェックリスト化したワンページサイトです。ブラウザだけで動き、チェック状態はローカルストレージに保存されます。

## できること
- デイリー 12 項目、ウィークリー 11 項目、週替わりショップ 7 項目をチェック管理
- 進捗カウンター表示（カテゴリ別）
- 「すべてリセット」「全項目を完了にする」ボタンで一括操作
- ローカルストレージに自動保存（同じブラウザなら保持）

## 使い方
- index.html をブラウザで開き、完了した項目にチェックを入れるだけで OK です。

## 公開（GitHub Pages）
1. GitHub でリポジトリを作成し、このフォルダの内容をコミットしてプッシュ。
2. GitHub → Settings → Pages で以下を設定。
   - Source: Deploy from a branch
   - Branch: main (または master) / (root)
3. 数十秒〜数分後に表示される URL で公開完了。

## ファイル
- index.html: 本体（チェックリスト + ローカルストレージ保存）
